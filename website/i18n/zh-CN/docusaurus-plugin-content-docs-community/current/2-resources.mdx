## 社区文档 {#community-documentation}

**[Docusaurus.community](https://docusaurus.community)** 网站通过更复杂的示例和完整文章扩展了[官方文档](/docs)，推荐最佳实践并提供常见问题的解决方案。

它也是一个完整的 Docusaurus 应用程序，就像我们的主站点一样，因此非常适合托管定制和实验示例。

:::info

如果社区认为指向这些文章的链接有用，我们可能会允许在官方文档中添加直接指向社区文档的链接，具体情况具体分析。

我们也考虑在将来进一步整合到官方文档中。

:::

## 社区插件 {#community-plugins}

### 搜索 {#search}

- [docusaurus-plugin-lunr](https://github.com/daldridge/docusaurus-plugin-lunr) - 用于创建本地搜索索引的 Docusaurus 插件，可与 Lunr.js 配合使用
- [docusaurus-lunr-search](https://github.com/lelouch77/docusaurus-lunr-search) - Docusaurus 的离线搜索
- [docusaurus-search-local](https://github.com/cmfcmf/docusaurus-search-local) - Docusaurus 的离线/本地搜索
- [@easyops-cn/docusaurus-search-local](https://github.com/easyops-cn/docusaurus-search-local) - Docusaurus 的离线/本地搜索（支持中文）
- [docusaurus-theme-search-typesense](https://github.com/typesense/docusaurus-theme-search-typesense) - 用于 [Typesense DocSearch](https://typesense.org/docs/latest/guide/docsearch.html) 的 Docusaurus 插件
- [@markprompt/docusaurus-theme-search](https://github.com/motifland/markprompt-js/tree/main/packages/docusaurus-theme-search) - 为 Docusaurus 添加生成式 AI / LLM 驱动的文档搜索插件，由 [Markprompt](https://markprompt.com) 提供支持。
- [meilisearch-docsearch](https://github.com/tauri-apps/meilisearch-docsearch) - 用于 [Meilisearch](https://www.meilisearch.com) 的 Docusaurus 插件
- [@orama/plugin-docusaurus](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus) - 用于 Docusaurus v2 的 [Orama](https://askorama.ai/) 插件
- [@orama/plugin-docusaurus-v3](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus-v3) - 用于 Docusaurus v3 的 [Orama](https://askorama.ai/) 插件
- [@getcanary/docusaurus-theme-search-pagefind](https://getcanary.dev/docs/integrations/docusaurus.html) - 创建 [Pagefind](https://pagefind.app/) 索引并使用 [Canary](https://github.com/fastrepl/canary) 作为 UI 基础组件。

### 集成 {#integrations}

- [docusaurus2-dotenv](https://github.com/jonnynabors/docusaurus2-dotenv) - 支持 dotenv 和其他环境变量的 Docusaurus 插件
- [posthog-docusaurus](https://github.com/PostHog/posthog-docusaurus) - 将 [PostHog](https://posthog.com/) 产品分析与 Docusaurus 集成
- [docusaurus-plugin-moesif](https://github.com/Moesif/docusaurus-plugin-moesif) - 添加 [Moesif API 分析](https://www.moesif.com/) 以跟踪用户行为并定位开发者在激活漏斗中的流失点
- [docusaurus-plugin-yandex-metrica](https://github.com/sgromkov/docusaurus-plugin-yandex-metrica) - 添加 [Yandex Metrika](https://metrika.yandex.ru/) 计数器用于评估网站流量和分析用户行为

### 功能增强 {#features}

- [docusaurus-theme-github-codeblock](https://github.com/saucelabs/docusaurus-theme-github-codeblock). 支持从公共 GitHub 仓库引用代码示例的 Docusaurus 插件
- [docs-to-pdf](https://github.com/jean-humann/docs-to-pdf) - 将文档生成 PDF 格式
- [docusaurus-plugin-sass](https://github.com/rlamana/docusaurus-plugin-sass) - 支持 Sass/SCSS 样式表
- [docusaurus-plugin-remote-content](https://github.com/rdilweb/docusaurus-plugin-remote-content) - 允许从远程源获取内容的 Docusaurus 插件
- [@graphql-markdown/docusaurus](https://github.com/graphql-markdown/graphql-markdown) - 用于从 GraphQL 模式生成文档的 Docusaurus 插件
- [docusaurus-plugin-react-docgen-typescript](https://github.com/atomicpages/docusaurus-plugin-react-docgen-typescript) - 从 `react-docgen-typescript` 加载类型注解的 Docusaurus 插件
- [docusaurus-plugin-react-docgen](https://github.com/atomicpages/docusaurus-plugin-react-docgen) - 从 `react-docgen` 加载类型注解的 Docusaurus 插件
- [docusaurus-plugin-module-alias](https://github.com/atomicpages/docusaurus-plugin-module-alias) - 快速为本地模块创建别名的 Docusaurus 插件
- [docusaurus-protobuffet](https://github.com/protobuffet/docusaurus-protobuffet) - 用于 Protobuf 合约文档的 Docusaurus 工具集
- [docusaurus-prince-pdf](https://github.com/signcl/docusaurus-prince-pdf) - 使用 PrinceXML 生成 PDF，支持更好的字体子集和目录功能。兼容 Docusaurus 站点
- [redocusaurus](https://github.com/rohit-gohri/redocusaurus) - 通过 [Redoc](https://github.com/redocly/redoc) 将 OpenAPI 文档集成到您的文档中的 Docusaurus 预设
- [plugin-image-zoom](https://github.com/flexanalytics/plugin-image-zoom) - Docusaurus 的图片缩放插件
- [docusaurus-plugin-typedoc](https://github.com/tgreyuk/typedoc-plugin-markdown/tree/master/packages/docusaurus-plugin-typedoc) - 使用 [TypeDoc](https://typedoc.org/) 构建文档的 Docusaurus 插件
- [docusaurus-openapi-docs](https://github.com/PaloAltoNetworks/docusaurus-openapi-docs) - 用于生成交互式 OpenAPI 文档的 Docusaurus 插件和主题
- [docusaurus-post-generator](https://github.com/moojing/docusaurus-post-generator) - 通过类似 `yarn gen-post new [模板] [文章名称]` 的命令快速添加博客/文档文件的命令行工具
- [docusaurus-graph](https://github.com/Arsero/docusaurus-graph) - 生成显示文档文件间链接关系的图谱视图的 Docusaurus 插件
- [docusaurus-i18n](https://github.com/moonrailgun/docusaurus-i18n) - 使用 openai 自动翻译 Docusaurus 文档
