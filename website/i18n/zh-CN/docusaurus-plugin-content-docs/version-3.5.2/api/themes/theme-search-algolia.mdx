# 📦 主题-search-algolia

该主题提供了一个与Algolia DocSearch轻松集成的`@theme/SearchBar`组件。结合`@docusaurus/theme-classic`使用，可实现便捷的搜索集成功能。更多信息请参阅[搜索](../../search.mdx)文档。

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

该主题还新增了可通过`/search`路径访问的搜索页面（作为可替换的`SearchPage`组件），并支持OpenSearch。您可以通过`themeConfig.algolia.searchPagePath`修改默认路径，设为`false`可禁用搜索页面。

:::tip

若已安装`@docusaurus/preset-classic`，则无需额外安装本主题作为依赖项。

:::
