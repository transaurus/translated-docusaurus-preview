# 📦 plugin-svgr

import APITable from '@site/src/components/APITable';

一个基于 [SVGR](https://react-svgr.com/) 的插件，可在构建时自动将 SVG 文件转换为 React 组件。

## 安装 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-svgr
```

:::tip

若您使用 `@docusaurus/preset-classic` 预设，则无需单独安装此插件作为依赖项。

您可以通过[预设选项](../../using-plugins.mdx#docusauruspreset-classic)配置此插件。

:::

## 配置 {#configuration}

可接受字段：

<APITable>

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `svgrConfig` | `object` | `{}` | The [SVGR config options](https://react-svgr.com/docs/options/), forwarded as is |

</APITable>

### 配置示例 {#ex-config}

您可通过插件选项配置此插件。

```js config-tabs
// Preset Options: svgr
// Plugin Options: @docusaurus/plugin-svgr

const config = {
  svgrConfig: {
    /* SVGR config */
  },
};
```
