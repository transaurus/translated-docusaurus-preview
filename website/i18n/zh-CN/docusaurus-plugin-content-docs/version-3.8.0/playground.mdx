# 演练场

演练场允许您**直接在浏览器中运行Docusaurus，无需安装任何依赖**！

其主要用途包括：

- 测试Docusaurus功能
- 提交问题报告

可使用易记的快捷链接 [docusaurus.new](https://docusaurus.new)。

请从下列选项中选择一个模板。

:::
