# 📦 plugin-rsdoctor

import APITable from '@site/src/components/APITable';

[Rsdoctor](https://rsdoctor.dev/) 插件可帮助您诊断 Docusaurus 站点的打包阶段问题，同时支持 Webpack 和 Rspack。

:::tip

使用该插件可定位拖慢打包速度的插件或加载器，集中精力优化性能瓶颈。

:::

## 安装 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-rsdoctor
```

## 配置 {#configuration}

可配置字段：

<APITable>

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `rsdoctorOptions` | `object` | `{}` | The [Rsdoctor bundler plugin options](https://rsdoctor.dev/config/options/options), forwarded as is |

</APITable>

### 配置示例 {#ex-config}

您可以通过插件选项配置此插件。

```js title="docusaurus.config.js"
export default {
  plugins: [
    [
      'rsdoctor',
      {
        rsdoctorOptions: {
          mode: 'lite',
        },
      },
    ],
  ],
};
```
