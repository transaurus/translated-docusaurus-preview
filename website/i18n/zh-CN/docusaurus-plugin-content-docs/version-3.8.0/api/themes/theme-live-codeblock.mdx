# 📦 主题-live-codeblock

该主题提供了一个由[react-live](https://commerce.nearform.com/open-source/react-live/)驱动的`@theme/CodeBlock`组件。您可以在[交互式代码编辑器](../../guides/markdown-features/markdown-features-code-blocks.mdx#interactive-code-editor)文档中了解更多信息。

```bash npm2yarn
npm install --save @docusaurus/theme-live-codeblock
```

### 配置 {#configuration}

```js title="docusaurus.config.js"
export default {
  plugins: ['@docusaurus/theme-live-codeblock'],
  themeConfig: {
    liveCodeBlock: {
      /**
       * The position of the live playground, above or under the editor
       * Possible values: "top" | "bottom"
       */
      playgroundPosition: 'bottom',
    },
  },
};
```
