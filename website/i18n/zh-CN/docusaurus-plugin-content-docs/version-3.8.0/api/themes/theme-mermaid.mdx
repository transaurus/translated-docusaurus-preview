# 📦 theme-mermaid

该主题提供了一个由 [mermaid](https://mermaid-js.github.io/) 驱动的 `@theme/Mermaid` 组件。您可以在[图表文档](../../guides/markdown-features/markdown-features-diagrams.mdx)中了解更多信息。

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

## 配置 {#configuration}

```js title="docusaurus.config.js"
export default {
  themes: ['@docusaurus/theme-mermaid'],
  // In order for Mermaid code blocks in Markdown to work,
  // you also need to enable the Remark plugin with this option
  markdown: {
    mermaid: true,
  },
};
```
