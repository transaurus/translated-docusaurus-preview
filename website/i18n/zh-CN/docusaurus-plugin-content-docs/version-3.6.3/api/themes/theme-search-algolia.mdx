# 📦 theme-search-algolia

该主题提供了一个与 Algolia DocSearch 轻松集成的 `@theme/SearchBar` 组件。与 `@docusaurus/theme-classic` 结合使用时，可实现非常便捷的搜索集成功能。更多信息请参阅[搜索](../../search.mdx)文档。

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

该主题还新增了可通过 `/search` 路径访问的搜索页面（可替换的 `SearchPage` 组件），并支持 OpenSearch。您可以通过 `themeConfig.algolia.searchPagePath` 修改默认路径，设为 `false` 可禁用搜索页面。

:::tip

如果您已安装 `@docusaurus/preset-classic`，则无需将其作为依赖项单独安装。

:::
