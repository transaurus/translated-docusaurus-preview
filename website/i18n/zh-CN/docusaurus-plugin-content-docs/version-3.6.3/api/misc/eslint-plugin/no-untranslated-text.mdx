# no-untranslated-text

import APITable from '@site/src/components/APITable';

强制要求JSX中的文本标签必须被翻译函数调用包裹。

当启用[i18n功能](../../../i18n/i18n-introduction.mdx)时，该规则确保网站上显示的所有标签都可翻译，避免字符串意外遗漏翻译。

## 规则详情 {#details}

**错误**代码示例：

```js
// Hello World is not translated
<Component>Hello World</Component>
```

**正确**代码示例：

```js
// Hello World is translated
<Component>
  <Translate>Hello World</Translate>
</Component>
```

## 规则配置 {#configuration}

可接受字段：

<APITable>

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `ignoredStrings` | `string[]` | `[]` | Text labels that only contain strings in this list will not be reported. |

</APITable>

## 禁用场景 {#when-not-to-use}

若未使用[i18n功能](../../../i18n/i18n-introduction.mdx)，可禁用此规则。对于明确不需要翻译的文本，也可禁用此规则。
