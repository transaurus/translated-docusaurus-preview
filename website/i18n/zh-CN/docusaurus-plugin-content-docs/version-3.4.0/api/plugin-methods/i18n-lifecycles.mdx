# 国际化生命周期

插件通过这些生命周期加载与国际化相关的数据。

## `getTranslationFiles({content})` {#getTranslationFiles}

插件声明其需要使用的JSON翻译文件。

返回翻译文件 `{path: string, content: ChromeI18nJSON}`：

- `path`：相对于插件本地化文件夹 `i18n/[locale]/[pluginName]` 的路径。应省略 `.json` 扩展名以保持通用性。
- `content`：使用Chrome i18n JSON格式。

这些文件将由 [`write-translations` CLI](../../cli.mdx#docusaurus-write-translations-sitedir) 写入插件的i18n子文件夹，并在调用 [`translateContent()`](#translateContent) 和 [`translateThemeConfig()`](#translateThemeConfig) 之前读取相应语言环境的文件。

示例：

```js title="my-plugin.js"
export default function (context, options) {
  return {
    name: 'my-plugin',
    // highlight-start
    async getTranslationFiles({content}) {
      return [
        {
          path: 'sidebar-labels',
          content: {
            someSidebarLabel: {
              message: 'Some Sidebar Label',
              description: 'A label used in my plugin in the sidebar',
            },
            someLabelFromContent: content.myLabel,
          },
        },
      ];
    },
    // highlight-end
  };
}
```

## `translateContent({content,translationFiles})` {#translateContent}

使用本地化的翻译文件翻译插件内容。

返回本地化的插件内容。

`contentLoaded()` 生命周期将使用 `translateContent()` 返回的本地化插件内容进行调用。

示例：

```js title="my-plugin.js"
export default function (context, options) {
  return {
    name: 'my-plugin',
    // highlight-start
    translateContent({content, translationFiles}) {
      const myTranslationFile = translationFiles.find(
        (f) => f.path === 'myTranslationFile',
      );
      return {
        ...content,
        someContentLabel: myTranslationFile.someContentLabel.message,
      };
    },
    // highlight-end
  };
}
```

## `translateThemeConfig({themeConfig,translationFiles})` {#translateThemeConfig}

使用本地化的翻译文件翻译站点 `themeConfig` 的标签。

返回本地化的 `themeConfig`。

示例：

```js title="my-plugin.js"
export default function (context, options) {
  return {
    name: 'my-theme',
    // highlight-start
    translateThemeConfig({themeConfig, translationFiles}) {
      const myTranslationFile = translationFiles.find(
        (f) => f.path === 'myTranslationFile',
      );
      return {
        ...themeConfig,
        someThemeConfigLabel: myTranslationFile.someThemeConfigLabel.message,
      };
    },
    // highlight-end
  };
}
```

## `async getDefaultCodeTranslationMessages()` {#getDefaultCodeTranslationMessages}

使用 `<Translate>` API的主题可以提供默认的代码翻译消息。

它应返回 `Record<string, string>` 格式的消息，其中键是翻译ID，值是根据站点当前语言环境本地化的消息（不包含描述）。

示例：

```js title="my-plugin.js"
export default function (context, options) {
  return {
    name: 'my-theme',
    // highlight-start
    async getDefaultCodeTranslationMessages() {
      return readJsonFile(`${context.i18n.currentLocale}.json`);
    },
    // highlight-end
  };
}
```
