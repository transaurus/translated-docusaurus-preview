# no-html-links

import APITable from '@site/src/components/APITable';

确保使用 Docusaurus 的 [`<Link>`](../../../docusaurus-core.mdx#link) 组件而非 `<a>` 标签。

`<Link>` 组件内置了预取(prefetching)和预加载(preloading)功能，同时支持构建时死链检测，并能帮助 Docusaurus 更好地理解站点结构。

## 规则详情 {#details}

该规则的**错误**代码示例：

```html
<a href="/page">go to page!</a>

<a href="https://x.com/docusaurus" target="_blank">X</a>
```

该规则的**正确**代码示例：

```js
import Link from '@docusaurus/Link'

<Link to="/page">go to page!</Link>

<Link to="https://x.com/docusaurus">X</Link>
```

## 规则配置 {#configuration}

可接受字段：

<APITable>

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `ignoreFullyResolved` | `boolean` | `false` | Set to true will not report any `<a>` tags with absolute URLs including a protocol. |

</APITable>
