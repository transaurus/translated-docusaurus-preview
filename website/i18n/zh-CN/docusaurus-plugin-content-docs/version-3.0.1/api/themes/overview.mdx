# Docusaurus 主题

我们提供官方 Docusaurus 主题。

## 核心主题 {#main-themes}

核心主题为 [文档](../plugins/plugin-content-docs.mdx)、[博客](../plugins/plugin-content-blog.mdx) 和 [页面](../plugins/plugin-content-pages.mdx) 插件提供用户界面实现。

- [@docusaurus/theme-classic](./theme-classic.mdx)
- 🚧 其他主题规划中

:::warning

目标是让所有主题具备完全相同的功能、用户体验和配置。

仅UI设计和底层样式框架会有所变化，您应能轻松切换主题。

目前尚未达成此目标：仅经典主题达到生产就绪状态。

:::
